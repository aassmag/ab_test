# Анализ A/B теста для интернет-магазина

## Задача
Оценить, приводит ли новое расположение кнопки "Купить" к статистически значимому росту конверсии.

## Данные
Использован синтетический/открытый датасет, содержащий:
- `user_id` - ID пользователя
- `group` - контрольная (A) или тестовая (B) группа
- `conversion` - факт покупки (1/0)
- `revenue` - сумма заказа

## Процесс анализа
1. **Предобработка данных:** Проверка на пропуски, аномалии.
2. **Проверка сплитования (A/A-тест):** Сравнение метрик групп до начала эксперимента.
3. **Расчет ключевых метрик:** Конверсия, средний чек, доход на пользователя (RPV).
4. **Статистическая проверка:** Z-тест для разницы долей (конверсия), t-тест для средних.
5. **Интерпретация результатов:** Анализ p-value, power теста.

## Результаты и выводы
- **Конверсия:** В группе B конверсия составила 12.5% против 11.7% в группе A.
- **Статистическая значимость:** P-value = 0.15 (> 0.05). **Различия не являются статистически значимыми.**
- **Рекомендация:** Отклонить гипотезу о влиянии изменения кнопки на конверсию. Для будущих тестов предложено увеличить длительность и размер выборки.
